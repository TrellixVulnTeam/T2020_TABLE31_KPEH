## How to Run

### Prerequisites

Install [Node.js](https://nodejs.org/en/)

### Step 1: Install Packages
```bash
npm install 
```

### Step 2: Start Application
```bash
npm start
```
Browser should auto open tab with running app.
If not, open browser and visit (http://localhost:3000/)
